# Backend Boilerplate (Node + Express + TypeScript)

## 🚀 Features

* Express + TypeScript ready setup
* Clean and scalable folder structure
* Environment configuration support
* Pre-installed core middlewares

---

## 📦 Pre-installed Core Packages

This template already includes the most commonly used backend dependencies:

* **express** → Web framework
* **cors** → Handle cross-origin requests
* **morgan** → HTTP request logging
* **helmet** → Security headers
* **cookie-parser** → Parse cookies
* **multer** → File uploads
* **dotenv** → Environment variables

👉 These are enough to start building most backend applications immediately.

---

## ⚙️ Setup

Install dependencies:

```bash
npm install
```

---

## ▶️ Run in Development

```bash
npm run dev
```

---

## 🏗️ Build

```bash
npm run build
```

---

## 🚀 Start Production Server

```bash
npm start
```

---

## 🔌 Optional Packages (Install as Needed)

Depending on your project requirements, you can install additional tools:

* **mongoose / prisma / sequelize** → Database (ODM/ORM)
* **bcryptjs** → Password hashing
* **jsonwebtoken** → Authentication (JWT)
* **zod / joi** → Validation

Example:

```bash
npm install mongoose bcryptjs jsonwebtoken
```

## 🧠 Philosophy

This boilerplate is designed to:

## 📁 Folder Structure Overview

* **config/** → App & DB configuration
* **controllers/** → Request handlers
* **services/** → Business logic
* **models/** → Database schemas
* **routes/** → API routes
* **middlewares/** → Custom middlewares
* **validators/** → Request validation
* **utils/** → Helper functions
